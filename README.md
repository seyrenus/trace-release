### About  jj is a Git-compatible version control system that is both simple and powerful. For example,   `colors."diff removed token" = { bg = "ansi-color-52", underline = false }`   will apply a dark red background on removed words in diffs.  ### Fixed bugs  * `jj file annotate` can now process files at a hidden revision.  * `jj op log --op-diff` no longer fails at displaying "reconcile divergent   operations."
