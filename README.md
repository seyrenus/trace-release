Back up your customized docker-compose YAML file (optional)     bash    cd docker    cp docker-compose.yaml docker-compose.yaml.$(date +%s).bak      2. Get the latest code from the main branch     bash    git checkout main    git pull origin main      3. Stop the service. Please execute in the docker directory     bash    docker compose down      4. Back up data     bash    tar -cvf volumes-$(date +%s).tgz volumes      5. Upgrade services     bash    docker compose up -d      ### Source Code Deployments  1. Stop the API server, Worker, and Web frontend Server.  2. Get the latest code from the release branch:     bash    git checkout 1.8.0      3. Update Python dependencies:     bash    cd api    uv sync      4. Then, let's run the migration script:     bash    uv run flask db upgrade      5. Finally, run the API server, Worker, and Web frontend Server again.  ---  ## What's Changed * test(api): fix flaky tests in TestWorkflowDraftVariableService by @QuantumGhost in
